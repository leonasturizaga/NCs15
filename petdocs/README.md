## Getting Started

1. Clone the repository:

```bash
    https://github.com/No-Country/S15-21-M-Python-Js.git
```

2. Install dependencies:

```bash
# with npm:
npm install
```

3. Run in your terminal:

```bash
# with npm:
npm run dev
```

and open http://localhost:5173
